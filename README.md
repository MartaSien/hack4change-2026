# Bałtyk Infokiosk

Projekt infokiosku edukacyjno-informacyjnego, składającego się z materiałów z recyklingu i ekranów eInk. Prezentuje informacje o gatunkach roślin i zwierząt zamieszkujących Bałtyk.

## Cele

Stworzenie żyjącego infokiosku bez energochłonnych ekranów LED, w miejscach "off the grid", takich jak plaże, lasy, tereny objęte ochroną. 

- niskie zużycie energii
- bez ciągnięcia kabli
- bez zanieczyszczenia światłem
